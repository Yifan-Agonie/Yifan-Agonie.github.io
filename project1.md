# Human-AI Interaction 

## Overview
This project explored how users interact with AI-assisted workflows.

## My Role
- UX Researcher
- Led user interviews and usability testing

## Methods
- User interviews
- Usability testing
- Behavioural analysis

## Key Findings
- Users struggled with trust in AI suggestions  
- High cognitive load in decision-making  

## Impact
- Improved task completion rate  
- Informed product design decisions  

## Reflection
This project strengthened my ability to translate research into actionable insights.
